A short story must have a single mood and every sentence must build towards it. ~Edgar Allan Poe

Now that you have thought about story goals, and scenes and sequels, let us look at the overall structure again. We said that we need a clear beginning, a middle, and an end, but we would like to equip you with a slightly more refined plot recipe, which you could use repeatedly.

Please note: this is not the only way to plan and write a short story. There are probably as many methods and plot structures out there as there are writers and stories.

However, this recipe works. If you follow this structure carefully, you will have a completed, rounded story with all the necessary elements.

It is as simple as this....

  1. Dramatic Start (Scene)

  2. Decisive Sequel

  3. Surprising Middle (Scene)

  4. Satisfying End (Scene)

Before we explore these four parts, there are still a couple of technical aspects to discuss.
